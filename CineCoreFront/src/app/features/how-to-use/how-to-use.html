How to Use CineCore

A simple guide to get started with the platform

01

Create an Account

Sign up to the platform by providing basic information. One account gives you access to all your projects.

02

Create a Project

After logging in, create a new film project: specify the title, genre, synopsis and planned start date. You'll automatically become the project owner.

03

Invite Your Team

Add participants to your project: managers, actors, crew members. Specify their roles and access permissions.

04

Upload Your Script

Import your screenplay text and mark it up: characters, lines, actions, locations and props. The system automatically creates databases for further work.

05

Organize Resources

Add locations with photos and addresses. Create a props list with acquisition types (purchase/rental) and status tracking.

06

Conduct Casting

Select actors for screenplay roles. Add candidates, review their profiles and make decisions.

07

Schedule Shooting

Create a shooting schedule: drag and drop scenes to calendar days, or use automatic planning. Generate call sheets for the team.

Key Features

Multi-tenant architecture: one login — many projects
Different roles: Owner, Manager, Actor, Crew
Interactive script markup (mobile-friendly)
Visual character color coding
AI-powered automatic shooting schedule
Call Sheet generation for teams
Centralized location and props management
Invitation and confirmation system
